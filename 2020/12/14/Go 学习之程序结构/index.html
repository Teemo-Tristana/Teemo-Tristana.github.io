Go 学习之程序结构

发表于 2020-12-14 更新于 2020-12-19

Go 程序结构

命名

  1. Go 语言中的函数名、变量名、常量名、类型名、语句标号和包名等所有的命名，都遵循一个简单的命名规则 : 以字母（Unicode字母）或下划线开头。

  2. Go 语言的 25 个关键字(关键字不能用于自定义名字，只能在特定的程序结构中使用)

  3. Go语言中，有 30 多个预定义的名字，主要是内置常量、类型和函数

    内部预定义的名字不是关键字,因此可以重新使用它们，但是尽量少重复使用，避免引起语义混乱。

  4. 函数名字的开头字母的大小写决定了该函数在包外的可见性，若是大写字母开头，则是可导出的，比如 fmt的Println()函数

  5. 推荐使用 驼峰式 命名

声明

Go 语言主要有四种类型的声明语句：

  • var 用于 变量

  • const 用于 常量

  • type 用于 类型

  • func 用于 函数

    变量

    包一级的声明可以对整个包有效，而函数内部的声明只能局部有效

  • 变量的声明方式

    1. var 变量名 类型 = 表达式
      如果初始化表达式被省略，那么将用零值初始化该变量。 数值类型变量对应的零值是0，布尔类型变量对应的零值是false，字符串类型对应的零值是空字符串，接口或引用类型（包括slice、指针、map、chan和函数）变量对应的零值是nil。数组或结构体等聚合类型对应的零值是每个元素或字段都是对应该类型的零值。
      零值初始化机制可以确保每个声明的变量总是有一个值，因此 Go 语言中不存在未初始化的变量。

    2. 多种类型同时声明

    3. 简短变量声明
      名字 := 表达式

指针

一个指针的值是另一个变量的地址。一个指针对应变量在内存中的存储位置。并不是每一个值都会有一个内存地址，但是对于每一个变量必然有对应的内存地址

任何类型的指针的零值都是 nil

Go 语言中返回函数局部变量的地址也是安全的

new函数

new 函数开业用于创建变量， new(T) 将创建一个T类型的匿名变量，并初始化为T类型的零值,并返回 变量地址(即指针), 返回类型是 T 类型的指针(T*)

用new创建变量和普通变量声明语句方式创建变量没有什么区别，换言之，new函数类似是一种语法糖，而不是一个新的基础概念。

变量的声明周期

变量的生命周期指的是在程序运行期间变量有效存在的时间段。对于在包一级声明的变量来说，它们的生命周期和整个程序的运行周期是一致的。而相比之下，局部变量的生命周期则是动态的：每次从创建一个新变量的声明语句开始，直到该变量不再被引用为止，然后变量的存储空间可能被回收。函数的参数变量和返回值变量都是局部变量。它们在函数每次被调用的时候创建。

Go 语言的自动垃圾收集器(基本思路)：
从每个包级的变量和每个当前运行函数的每一个局部变量开始，通过指针或引用的访问路径遍历，是否可以找到该变量。如果不存在这样的访问路径，那么说明该变量是不可达的，也就是说它是否存在并不会影响程序后续的计算结果

编译器会自动选择在堆或栈上分配局部变量的存储空间，但是，并不是由 var 或 new 声明方式决定的，有编译器决定。

逃逸的变量需要额外分配内存，同时对性能的优化可能会产生细微的影响。

类型

变量或表达式的类型定义了对应存储值的属性特征

type 类型名字 底层类型

包 和 文件

Go 语言中的 包 类似其他语言的模块或库，目的是为了支持模块化、封装、单独编译和代码复用。

数据类型

Go 语言将数据类型分为四类 : 基础类型、复合类型、引用类型和接口类型

基础类型

基础类型比如 数字、字符串、布尔等类型

  • 整数类型

    • Go 提供了四种不同大小的整数类型: int8/int16/int32/int64 和 uint8/uint16/uint32/uint64。

    • 也提供了特定的CPU的 int 和 uint(32bit/64bit),具体大小与编译器有关。

    • Go采用的是 Unicode 字符，UTF8 编码， 它的 rune 类型和 int32 等价，表示一个 Unicode 码点

    • byte 和 uint8 等价，一般用于数值。

    • 不管它们的具体大小，int、uint和uintptr是不同类型的兄弟类型。其中int和int32也是不同的类型，即使int的大小也是32bit，在需要将int当作int32类型的地方需要一个显式的类型转换操作。

  • 浮点数类型

    • Go语言提供了两种精度的浮点数，float32和float64

  • 复数
    Go语言提供了两种精度的复数类型：complex64和complex128，分别对应float32和float64两种浮点数精度。内置的complex函数用于构建复数，内建的real()和imag()函数分别返回复数的实部和虚部

  • 布尔类型

    • 一个布尔类型的值只有两种：true和false。if和for语句的条件部分都是布尔类型的值，并且==和<等比较操作也会产生布尔型的值。

  • 字符串

    • 一个字符串是一个不可改变的字节序列(字符串是只读)。字符串可以包含任意的数据，包括byte值0，但是通常是用来包含人类可读的文本。文本字符串通常被解释为采用UTF8编码的Unicode码点（rune）序列。
    • len() 返回字符串的字节数目(而不是 rune 字符数目)
    • 标准库中有四个包对字符串处理尤为重要：bytes、strings、strconv和unicode包
      • strings 提供字符串相关操作比如替换、查询、比较等
      • bytes 功能和 strings 包类型，但是只要是争对 []byte 类型
      • strconv包提供了布尔型、整型数、浮点数和对应字符串的相互转换，还提供了双引号转义相关的转换。
      • unicode包提供了IsDigit、IsLetter、IsUpper和IsLower等类似功能，它们用于给字符分类

  • 常量

    • 常量表达式的值在编译期就已经确定了，所有常量的运算都可以在编译期完成，这样可以减少运行时的工作，也方便其他编译优化
    • iota
      • 在一个const声明语句中，在第一个声明的常量所在的行，iota将会被置为0，然后在每一个有常量声明的行加一。

复合类型

复合类型比如数组和结构体

数组

数组是一个由固定长度的特定类型元素组成的序列，一个数组可以由零个或多个元素组成，一旦声明了，数组的长度就固定了，不能动态变化。

len() 和 cap() 返回结果始终一样。

结构体

结构体是将零个或多个任意类型的变量，组合在一起的聚合数据类型，也可以看做是数据的集合。

引用类型

数组、结构体的组合

接口类型

指针、切片、字典、函数、通道

指针

切片

切片是一种动态数组，比数组更为灵活，长度不固定，可以进行追加和删除等，类似C++中的vector

字典map

map 集合是无序的 key-value数据结构

map 集合中的 key 和 value 可以是任意类型，但所有的 key 必须属于同一数据类型，所有value 也必须是同一类型， key 和 value 的类型可以不同

函数

  1. defer 函数 ：defer 声明的函数不会立即去执行，而是在函数 return 后再去执行，主要用在 异常处理、日志记录、清理数据、释放资源的等场景
